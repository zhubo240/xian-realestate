# 西安房地产数据集

西安市小区级房地产数据采集与可视化项目。

## 项目目标

采集西安所有小区的基础数据（名称、位置、均价、户型、年龄等），并在地图上展示。

## 技术方案

- **数据源**：房天下 (fang.com)
- **经纬度**：高德地图 API（免费）
- **存储格式**：CSV
